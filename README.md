- 1과 10중 하나의 수 랜덤으로 선정한다.

범위에서 벗어난 수를 입력 시, 다시 범위를 알려준다.
또한 정답을 맞췄을 때 게임 재시작 여부를 입력한다.
y를 누르면 랜덤의 수를 다시 설정하고, n을 입력 시 게임이 종료된다.
